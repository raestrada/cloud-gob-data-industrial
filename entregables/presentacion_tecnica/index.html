Cluster Linking

  • Replicación async edge → cloud
  • Latencia: <50ms + geo ~35ms
  • Exactly-once semantics

Tiered Storage

  • Cloud: 30d hot + ∞ cold (GCS)
  • Edge: 7d local buffer
  • Ahorro: 50-70% vs hot-only

Decisiones de Arquitectura Consensuadas

1. RPO/RTO=0 Interpretación

RPO/RTO=0 = operación local (planta). DR negocio: RPO=segundos OK (Cluster Linking).

2. Medallion 4 Capas

BRONZE en edge (reduce tráfico WAN 60-70%). SILVER/GOLD en cloud (analítica multi-planta).

3. Confluent Cloud vs Self-Managed

Cloud en GCP (hub/dr). Platform self-managed en Edge (autonomía). Cluster Linking conecta.

4. Dual Interconnect 2x1Gbps

Requisito indispensable para throughput 50-150 Mbps/planta + redundancia.
